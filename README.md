# FIFA 2018 Scores CLI

Code Inspired from https://github.com/johnbfox/world-cup-scores-cli (I clone and Modified this for Learning Purpose)

## Installing

```sh
git clone https://github.com/mskian/fifaupdate
```

## Install NPM

```sh
npm install
```

## Link NPM and Bash

```sh
sudo npm link
```

## Runnling

```sh
fifascore
```
